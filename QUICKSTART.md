# Helldivers Numpad Macros - Quick Start

## Installation

### Installing from the Installer (Recommended)

1. Download `HelldiversNumpadMacros-Setup-v1.0.0.exe` from the latest release
2. Run the installer
3. Follow the installation wizard
4. The application will be installed to `C:\Program Files\Helldivers Numpad Macros`
5. Start menu and desktop shortcuts will be created (if selected)

### Portable Version

1. Download `HelldiversNumpadMacros-Portable-v1.0.0.exe` from the latest release
2. Place it in any folder
3. Run the executable directly (no installation needed)
4. Profiles and settings are stored in the same folder as the executable

## First Run

On first run, the app may request administrator privileges if enabled in settings. This is required if Helldivers 2 runs with admin rights.

## Troubleshooting

**Macros not working:**

- Ensure "Status: Enabled" at the bottom
- Check if administrator privileges are needed (Settings)
- Verify latency setting (try increasing to 30-40ms)

**Can't find the app after minimizing:**

- Check system tray (bottom-right corner)
- Double-click tray icon to restore window

**Update check fails:**

- Check internet connection
- GitHub may be temporarily unavailable
